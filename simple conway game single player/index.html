Rules of the Game:


For a space that is 'populated':
Each cell with one or no neighbours dies, as if by solitude.
Each cell with four or more neighbours dies, as if by overpopulation.
Each cell with two or three neighbours survives.


For a space that is 'empty' or 'unpopulated'
Each cell with three neighbors becomes populated.


Make a pattern yourself by clicking on the cells with the mouse.
A new generation of cells (corresponding to one iteration of the rules) is initiated automatically.
In case, server is present, the game can be played by multiple players( Tested on more than 100 chrome instances simulataneously).
